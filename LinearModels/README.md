#Welcome to Linear Models

These 4 tutorials walk you through the basics of linear models, from simple linears models through to more complex generalised or mixed effects models. It's all in a frequentist framework, you could do similar things in a [Bayesian](https://ourcodingclub.github.io/tutorials/brms/) framework. Here are a few other resources you can lean on to help you:

 - Graph Pad [“The Ultimate Guide to Linear Regression”](https://www.graphpad.com/guides/the-ultimate-guide-to-linear-regression) (start here if everything else bamboozles you, it’s code free and they explain things great!)
 - Tim Newbold’s [excellent blog](https://timnewbold.github.io/teaching_resources/GLMs.html) about GLMs (and the more advanced on on [mixed effects models](https://timnewbold.github.io/teaching_resources/MixedEffectsModels.html) )
 - Frans Rodenburgs [amazing videos](https://fransrodenburg.github.io/Applied-Statistics/GLM.html)
